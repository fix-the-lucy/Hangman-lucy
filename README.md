# üéÆ How to Play
1. The computer selects a secret word.
2. You must guess one letter at a time.
3. If the letter is in the word, the blanks fill in.
4. If the letter is wrong, the hangman starts to appear.
5. *Win:* Reveal all letters before the hangman is complete.
6. *Lose:* Fail to guess the word within the allowed number of attempts.
